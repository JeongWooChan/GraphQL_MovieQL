## 1. 환경 설정 

### 1. graphql-yoga 설치 

```bash
yarn add graphql-yoga 
```

### 2. nodemon 설치 

```bash
yarn global add nodemon
```

### 3. babel 설치 

```bash
yarn add babel-node --dev
yarn global add babel-cli
yarn add babel-cli babel-preset-env babel-preset-stage-3 --dev
```

### 4. node-fetch 설치 

```bash
yarn add node-fetch 
```
